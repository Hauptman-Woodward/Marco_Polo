Bug Reports and Suggestions
===========================

If you have found a bug in Polo, error in the documentation or anything
along those lines please report it! If you have some GitHub knowhow you can report it on the issues
`on the issues <https://github.com/Hauptman-Woodward/Marco_Polo/pulls>`_
page of the Polo repository or fill out the Google form below.
