Welcome to Polo’s documentation!¶

Polo is a python GUI build using the PyQt library for high-throughput crystallization screening users. It was created by Ethan Holleman for the 2020 BioXFEL summer internship program. Funding provided through NSF BioXFEL STC Grant Number 1231306, NSF BIO 2029943, NIH NIGMS 124135.
